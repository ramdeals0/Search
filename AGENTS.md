# AGENTS.md

## Product
This repository contains a retailer search and discovery platform MVP.

## Phase 1 objective
Ship a working search platform with:
- catalog ingestion
- product search
- autocomplete
- filters
- merchandiser rules
- event tracking
- basic analytics

## Priority order
1. Working search over perfect architecture
2. Clean schema over clever ranking
3. Data quality over advanced AI
4. Instrumentation before optimization

## Tech preferences
- TypeScript
- Next.js
- Node.js
- Postgres
- OpenSearch-compatible architecture
- Shared types and schemas

## Developer expectations
- Make minimal safe changes
- Keep file structure clean
- Document new commands
- Ask before introducing major dependencies
